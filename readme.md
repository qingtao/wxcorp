# 微信企业号应用和小程序

## 1. 企业号增加小程序方法

- "应用与小程序" - "应用" - "创建应用";
- 填写"应用名称"和应用介绍(选填)", 选择部门和成员，以上内容都可在创建后修改;
- 应用等同于微信的服务号;
- 后台程序需要"AgentId"和"Secret"值;

## 2. 服务端接口

- 协议: HTTPS
- 请求数据格式: JSON
- 字符编码: UTF8

## 2. access_token

- 令牌有有效期，过期需刷新
